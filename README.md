This project is an end-to-end Exploratory Data Analysis (EDA) that cleans, processes, and visualizes an administrative police incident and Use of Force (UOF) dataset using R. Heavily aligned with data-driven transparency frameworks, the study systematically cleans raw operational logs, standardizes temporal and geographic entries, and evaluates demographic data across law enforcement personnel and involved subjects. Through interactive statistical plots, geospatial coordinate mapping via Leaflet, and a Chi-Square independence analysis, the project identifies critical trends regarding crime fluctuations, operational tenure correlations, injury distributions, and regional incident hotspots. Ultimately, it transforms raw municipal records into actionable data visualizations aimed at uncovering behavioral root causes and informing fair, equitable public safety practices.
